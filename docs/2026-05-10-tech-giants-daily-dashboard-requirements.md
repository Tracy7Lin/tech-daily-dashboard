# 科技巨头每日情报看板 v1 需求文档

## 1. 项目概述

`科技巨头每日情报看板` 是一个面向个人使用的静态日报站点。系统按北京时间每天自动聚合中美科技巨头的高价值官方动态，生成中文速览摘要，并在多家公司围绕相近主题发布信息时给出横向对比。

该产品的目标不是做新闻搬运，而是产出一份可快速扫读、可回溯、带有趋势判断的每日情报面板。

## 2. 产品目标

- 每天自动生成一份科技巨头日报网页
- 聚焦高价值官方动态，而不是全网新闻抓取
- 将原始内容统一转化为中文速览摘要
- 自动识别跨公司的相似主题并生成比较结论
- 保留历史归档，便于按日期回看

## 3. 目标用户

- 个人使用者
- 重点场景是每天快速浏览科技行业动态与趋势，而非团队协作或公开媒体发布

## 4. 观察名单

首版采用固定默认名单，后续保留可配置扩展能力。

### 美国 10 家

- OpenAI
- Google
- Microsoft
- Anthropic
- Apple
- Meta
- Amazon
- NVIDIA
- Tesla
- xAI

### 中国 5 家

- Tencent
- ByteDance
- Alibaba
- Huawei
- Xiaomi

当前优先级以软件、AI 平台、云服务、终端生态与互联网平台为主。芯片与算力公司暂不单独提升为重点方向，但保留后续扩展空间。

## 5. 内容范围

允许收录的内容类型：

- AI / 大模型 / Agent / 开发者能力
- 新产品、新服务、新平台能力
- 云、协作、企业软件、生态开放
- 组织方向、战略合作、未来展望

默认降权或过滤的内容：

- 纯营销宣传
- 无实质信息的品牌内容
- 招聘宣传
- 不具备实际产品或战略信息的活动预告

## 6. 信息源边界

首版采用 `官方优先` 策略，只使用可验证的官方来源，例如：

- 官方博客
- 官方 newsroom / press release 页面
- 官方开发者博客
- 官方产品更新页
- 官方公告页
- 必要时纳入官方社媒账号链接作为补充入口

产品目标偏 `高准确率`，不追求全网覆盖。

## 7. 生成口径

- 每天按 `北京时间自然日` 生成一份日报
- 采用 `全自动` 流程
- 当天若某公司没有有效动态，应明确展示“今日无有效动态”
- 若某个官方源临时不可用，应显示源状态或抓取失败提示，而不是静默跳过
- 对依赖文章详情页提取发布时间的 HTML 信源，如果无法拿到可信发布时间，则该条内容默认不进入当日日报，以保证日报日期边界可信

## 8. 输出形态

首版产出为静态网页站点，至少包含两层：

- 首页：当天总览、主题对比、公司动态摘要
- 每日日报详情页：当天完整内容

同时提供按日期浏览的历史归档入口。

## 9. 页面结构

### 9.1 今日总览

- 日期
- 今日有效新闻数
- 覆盖公司数
- 今日最热主题
- 一句话趋势判断

### 9.2 主题对比

- 按主题聚合当天最值得关注的 2-5 个方向
- 每个主题展示相关公司、差异点、趋势判断、原文入口

### 9.3 公司动态

- 按公司展示当天新闻摘要
- 同公司多条内容按重要度优先，再按时间排序

### 9.4 历史归档

- 按日期查看过往日报
- 首版不要求复杂搜索，只要求可稳定回看

## 10. 摘要要求

每条新闻都需要生成中文总结，而不是直接转载原文。摘要默认控制在 `1-3 句`，并统一回答：

- 发生了什么
- 为什么重要
- 它更偏产品动作、技术动作还是战略动作

对于特别重要的动态，可以补一句影响判断，但整体仍以速览为主，不做长篇解读。

## 11. 对比要求

对比以 `主题` 为核心，而不是固定公司对公司比较。系统应自动发现相似主题，例如：

- AI agent
- 新模型发布
- 企业 AI 平台
- 开发者生态
- 终端 AI 入口

每个对比块至少应说明：

- 哪几家公司在做相近的事
- 各自切入点的差异
- 谁偏产品落地，谁偏基础能力，谁偏生态布局
- 这反映出的行业趋势

## 12. 质量要求

- 宁可少报，不要高噪音
- 所有摘要都应可追溯到官方原文
- 页面以快速理解为优先，不追求内容堆量
- 首版重点验证内容选择、摘要质量、主题对比质量，而不是复杂交互

## 13. 可扩展性要求

虽然 v1 是静态站点，但需求上应保留后续扩展空间：

- 公司名单可配置扩容
- 信源列表可扩展
- 主题体系可调整
- 后续可升级为支持筛选、检索、标签浏览的动态看板
- 不应将首版设计成一次性日报脚本

## 14. 非目标

首版不包含以下能力：

- 用户登录与权限系统
- 团队协作
- 评论、收藏、个性化订阅
- 全网媒体级新闻聚合
- 深度研究报告式长文分析
